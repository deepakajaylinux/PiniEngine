PiniEngine

PiniEngine은 비주얼노벨 게임 제작에 특화된 게임엔진 입니다.

PiniEngine을 사용하게 되신것을 환영합니다.

비주얼노벨 게임 제작에 특화된 게임엔진 입니다. 간단하게 스크립트를 써내려가는 것 만으로 쉽게 비주얼노벨(사운드노벨)류의 게임 제작이 가능합니다. 이후 추가될 플러그인들을 통해서 다양한 장르의 게임으로 확장도 가능합니다.
(제작 완료 단계에 있으며, 2014년 중으로 완전히 공개 될 예정입니다. )

라이센스

무료게임 제작과 상용게임 제작 모두 무료로 엔진을 사용 할 수 있습니다.

주요 특징

  • 뛰어난 스크립팅 환경

    • 피니엔진에서는 아래와 같은 스크립트 작성 환경을 제공합니다. 흐름제어와 기본적인 변수 사칙연산등이 가능합니다.

  • 한글 스크립트

    • 명령어가 모두 한글로 되어있어 작성이 쉽고 가독성이 높습니다.

  • 코드 하이라이팅

    • 스크립트의 가독성을 위해 예약어 하이라이팅을 지원합니다.

  • 에러 표시

    • 스크립트 에디팅 영역에서 문법 에러를 표시합니다.

  • 명령어 자동완성 & 스크립팅중 미리보기

    • 명령어를 편리하게 불러와 사용 할 수 있도록 명령어 자동완성 기능을 지원합니다.
    스크립트를 작성하는 동안에 게임화면에서 어떻게 보일지 미리 확인 할 수 있습니다.
    매 순간 문체를 다듬는데 집중 할 수 있습니다.

PiniEngine에 대한 모든 저작권은 nooslab에 있습니다.
