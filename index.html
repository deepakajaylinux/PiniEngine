PiniEngine

PiniEngine은 비주얼노벨 게임 제작에 특화된 게임엔진 입니다.

PiniEngine을 사용하게 되신것을 환영합니다.

비주얼노벨 게임 제작에 특화된 게임엔진 입니다. 간단하게 스크립트를 써내려가는 것 만으로 쉽게 비주얼노벨(사운드노벨)류의 게임 제작이 가능합니다. 이후 추가될 플러그인들을 통해서 다양한 장르의 게임으로 확장도 가능합니다.
(제작 완료 단계에 있으며, 2014년 중으로 완전히 공개 될 예정입니다. )

라이센스

무료게임 제작과 상용게임 제작 모두 무료로 엔진을 사용 할 수 있습니다.

주요 특징

  • 뛰어난 스크립팅 환경

    • 피니엔진에서는 아래와 같은 스크립트 작성 환경을 제공합니다. 흐름제어와 기본적인 변수 사칙연산등이 가능합니다.

  • 한글 스크립트

    • 명령어가 모두 한글로 되어있어 작성이 쉽고 가독성이 높습니다.

  • 코드 하이라이팅

    • 스크립트의 가독성을 위해 예약어 하이라이팅을 지원합니다.

  • 에러 표시

    • 스크립트 에디팅 영역에서 문법 에러를 표시합니다.

  • 명령어 자동완성 & 스크립팅중 미리보기

    • 명령어를 편리하게 불러와 사용 할 수 있도록 명령어 자동완성 기능을 지원합니다.
    스크립트를 작성하는 동안에 게임화면에서 어떻게 보일지 미리 확인 할 수 있습니다.
    매 순간 문체를 다듬는데 집중 할 수 있습니다.

    피니엔진 스크립팅 환경


멀티 플랫폼

  • 제작 환경

    • 다양한 환경에서 배포와 제작이 가능합니다. 엔진에 포함된 각 툴은 모두 윈도우즈와 맥에서 사용이 가능합니다.

  • 크로스 플랫폼 배포

    • 피니엔진으로 제작된 게임은 다음 플랫폼으로 배포가 가능합니다.

    • Windows
      • 피니엔진을 이용해 CD로 구워 배포할 수 있습니다.
      • 구워진 CD를 PC에 넣으면 자동으로 설치프로그램이 시작 됩니다.
      • 자동실행되는 인스톨러 화면도 제작자가 편집 할 수 있습니다. 오프라인을 통해 판매를 하려고 할 때 유용하게 사용 할 수 있습니다.
    • Web
      • 플래시를 통해서 웹으로 배포 할 수 있습니다. 데모버젼을 웹상에 공개하여 제작된 게임을 홍보하는데 사용 할 수 있습니다.
    • Android
      • 국내 점유율의 많은 비율이 안드로이드 마켓입니다. 안드로이드 마켓에 게임을 서비스하여 수익을 낼 수 있습니다.
    • iOS
      • 애플 앱스토어에 게임을 올릴 수 있습니다. 단 애플 디바이스(아이폰, 아이패드)를 위해서는 아래 조건들을 만족하는 개발환경이 갖춰져야 합니다. 위 조건과 장비를 구비하는 것이 부담스러운 분들을 위해서 채널링을 진행 하고 있습니다. 자세한 내용은 '채널링'항목을 참고해주세요
        • MacOS Snow leopard
        • XCode 최신버젼
        • Apple 개발자 등록 계약
